Yosemite正式版发布以后，5G的安装包升级了，记录一下升级的问题和注意事项：

备份

备份不一定可以找一个移动硬盘，Time Machine可以选择备份哪些资料，比如Downloads这种巨大又没太大用的目录就不要备份了，占硬盘还慢。

安装完成

没来及享受新系统带来的欣喜，发现问题了：

  1. brew出问题了，运行brew运行出错
    • » mv /usr/local/Cellar /tmp/
    • » ruby -e "$(curl -fsSL https://raw.github.com/mxcl/homebrew/go/install)"重新安装homebrew
    • » brew update
    • » brew doctor根据提示修复环境
    • » brew upgrade
  2. brew upgrade时vim报错
  3. brew upgrade时subversion报错
  4. brew upgrade时php55报错
    • 因为执行brew doctor之后建议我安装xcode6.1，我没有理会导致前面三个东东编译失败。
    • App Store里是找不到xcode6.1的，需要到这里找传送门，有2.5G之巨
    • 安装xcode后, brew upgrade就妥了
  5. java -version提示更新java，更新就是了。

设置出错

因为apache是系统自带的，这回顺便也把apache给我干掉了，配好的virtual host也不行了。

检查了virtual host的端口已经启来了，document root的读写权限apache的User(_www)也都有。

解决办法：vi /etc/apache2/extra/httpd-vhosts.conf

到apache的配置文件重新更改extension的路径：

  • » LoadModule php5_module /usr/local/Cellar/php55/5.5.18/libexec/apache2/libphp5.so
  • » sudo apachectl restart

Spotlight

到偏好设置里把Bing Web Searches和Spotlight Suggestions速度关掉，每次用它还偷偷告诉水果和巨硬我的隐私，你妹啊！

提示一下小技巧

  • 更改环境变量export PATH=/usr/local/bin:$HOME/bin:$PATH(注意顺序)
  • 如果使用phpstorm，在设置里面搜subversion，在use command line client栏中写: /usr/local/bin/svn才行，估计是调用的时候前面加了\

Enjoy

  • 手机升级到ios8以后，手机和电脑在一个局域网中就可以实现在电脑上接电话，也可以让手机连接电脑分享出来的热点。我决不抱怨屌丝北邮上网终端限制太蛋疼
  • handoff，只要手机和电脑蓝牙连接起来，就可以实现了，现在只有apple自家的程序支持，我的Wunderlist?Day one?Skype?QQ?We chat?他们还需要时间。
  • 大赞升级以后iterm2全盘的改变，热键把iterm2切出来的时候干嘛每次要那么多过场动画，就现在这样在当前的space全屏多nice，喜欢透明背景的童鞋还能装装B。
    • 看起来比原来要舒服，过去会偶尔无法进入MissionControl界面，现在改善了。
  • 新的通知栏和Safari的改进，确实很棒，解决了蛋疼的问题，谁用谁知道。
